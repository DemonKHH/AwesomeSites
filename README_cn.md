## 如何使用

</div>

每个收集的网站都经过测试和修复，都可以在本地主机上运行。克隆存储库后，进入网站的文件夹，只需启动本地 HTTP 服务器（例如`live-server`）即可离线运行网站。

示例
```bash
clone git@github.com:ezshine/AwesomeSites-Pack01.git
cd ./AwesomeSites-Pack01/game/choochooworld.com/
live-server
```
